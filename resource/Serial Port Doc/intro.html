Introduction

History

Qt Serial Port originated from the third-party library QSerialDevice 2.0.

Supported platforms

The primary goal is to support as many common platforms as possible. Please find the information below about the support state for the various platforms:

PlatformSupport stateNotes
Windows x86/x64 NT based (2K, XP, Vista, 7, 8)Complete support (Tier 1).Support expected for Windows 8, but has not yet been tested.
Windows CEPartial support (Not Tier 1).Tested only with platforms 5 and 6 by using the emulator.
SymbianPartial support, and only for legacy Qt4 (Not Tier 1).The implementation is not complete and further support in doubt.
Gnu/LinuxComplete support (Tier 1).
Mac OSXComplete support (Tier 1).
Other *nixComplete support (Tier 1).All POSIX-compatible.
