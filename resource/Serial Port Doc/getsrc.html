Users

Developers

Getting a Qt developer account is necessary to contribute to the project, so that has to be obtained first. Once, that is done, see the following page for further details about the Git and Gerrit usage:

Setting up Gerrit

The following command can be used to clone the project with push access:

$ git clone ssh://<username>@codereview.qt-project.org:29418/playground/qtserialport.git
